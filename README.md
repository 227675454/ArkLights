## 特点

1. 支持明日方舟官服与B服。
1. 支持常见真机、云手机及模拟器（安卓7至11，DPI>=320，分辨率>=720x1280，长宽比>=16:9）。
1. 支持热更新。免root。极低等待耗时，快过手操。
